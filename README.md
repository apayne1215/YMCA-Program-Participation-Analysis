# YMCA-Program-Participation-Analysis

## Overview
This project analyzes mock youth program participation data to identify attendance trends, retention patterns, and opportunities to improve program engagement.

## Tools Used
- Python
- pandas
- matplotlib

## Key Questions
- Which programs have the highest attendance rates?
- How does attendance change over time?
- How long do participants remain engaged after enrollment?

## Key Findings
- Programs such as After School Care and Youth Sports showed higher attendance and retention.
- Attendance trends varied seasonally, with changes during the beginning and end of the summer months and the school year.
- Retention analysis highlighted differences in engagement duration across programs.

## Notes
All data used in this projects is mock data created for educational purposes.
